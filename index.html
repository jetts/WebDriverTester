Microsoft WebDriver Test Page

WebDriver is an emerging W3C standard through which web developers can write tests to automate testing their sites through web browsers. The full list of commands implemented in Microsoft Edge can be found here.

Download the appropriate Microsoft WebDriver server below:

  • WebDriver for Windows Insiders
  • Microsoft WebDriver Fall 2015 Update
  • Microsoft WebDriver

WebDriver Command

Send Request

Host

The host to send the command to.

Port

The port to send the command to.

Command

The command to be sent to Microsoft Edge and control the browser. Below the dropdown is a quick summary of what the command does.

Method

The HTTP method used for each command. Every command will either be a GET, POST or DELETE.

Path

The path of the command selected. This is used by Microsoft WebDriver to determine the command. Any commands not in the supported commands list will get a "command not implemented" error response.

Request Body

This is the body of the request being sent to the Microsoft WebDriver server. Some commands will have values that you can add here, others do not require a request body.
